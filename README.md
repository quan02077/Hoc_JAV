# 📚 Học JavaScript

Repository lưu trữ các bài tập và dự án học tập JavaScript.

## 📖 Nội dung

- Các bài tập cơ bản về JavaScript
- Ứng dụng thực hành ES6+
- Hàm, đối tượng, mảng, và xử lý DOM

## 🛠️ Công nghệ sử dụng

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
- **Bootstrap** (nếu có)

## 🚀 Cách sử dụng

1. Clone repository
2. Mở file HTML bằng trình duyệt
3. Xem kết quả trên DevTools Console

## 📝 Ghi chú

Repository được tạo nhằm mục đích học tập và ôn luyện JavaScript.
